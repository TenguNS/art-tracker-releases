## Что это

Маленькое настольное приложение для Windows, которое живёт в трее и автоматически
ведёт статистику вашей работы над артом: сколько времени вы реально рисовали,
сколько раз сохраняли файл, как растёт детализация. Помогает понять свой ритм
и обоснованно назвать цену за работу.

## Возможности

- ⏱ **Учёт чистого времени** — считается, только когда окно редактора активно
- 💾 **Сохранения и слои** — фиксирует Ctrl+S, размер и число слоёв (PSD/CLIP)
- 📊 **Календарь активности** за 60 дней со сводкой
- ₽ **Рекомендуемая ставка** — на основе вашей статистики
- 📤 **Отчёты PNG/PDF** по любой работе — для клиента или портфолио
- 🎨 **Темы** (тёмная/светлая) + любой акцентный цвет
- 🌍 **Русский / English**
- 🖌 Поддержка **Photoshop, Live2D Cubism, Clip Studio, SAI, Krita** и др.

## 🔒 Приватность

Всё хранится **только на вашем компьютере**. Никаких скриншотов экрана, никакого
облака, никаких аккаунтов — приложению просто некуда отправлять данные.

## Установка

1. Скачайте **ArtTracker-Setup.exe** (кнопка выше) и запустите
2. Windows покажет «Неизвестный издатель» → **Подробнее → Выполнить в любом случае**
   *(приложение не подписано платным сертификатом — это нормально)*
3. Укажите папку с рисунками — и просто работайте

Есть и **портативная версия** (`.zip`) на [странице релизов](https://github.com/TenguNS/art-tracker-releases/releases/latest) — распаковал и запустил, без установки.

## Обновления

Приложение само проверяет новые версии: **Настройки → Проверить обновления** →
скачать и установить в один клик.
