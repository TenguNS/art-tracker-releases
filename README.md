# 🎨 Art Tracker

**Тихий трекер продуктивности для цифровых художников и Live2D-риггеров.**
Сам считает время рисования, сохранения и слои — без скриншотов и облака.

</div>

---

## Что это

Маленькое настольное приложение для Windows, которое живёт в трее и автоматически
ведёт статистику работы над артом: сколько времени вы реально рисовали, сколько раз
сохраняли файл, как растёт детализация. Помогает понять свой ритм и обоснованно
назвать цену за работу.

## Возможности

- ⏱ Учёт чистого времени — считается, только когда окно редактора активно
- 💾 Сохранения и слои — фиксирует Ctrl+S, размер и число слоёв (PSD/CLIP)
- 📊 Календарь активности за 60 дней со сводкой
- ₽ Рекомендуемая ставка на основе статистики
- 📤 Отчёты PNG/PDF по любой работе
- 🎨 Темы (тёмная/светлая) + любой акцентный цвет
- 🌍 Русский / English
- 🖌 Photoshop, Live2D Cubism, Clip Studio, SAI, Krita и др.

## 🔒 Приватность

Всё хранится только на вашем компьютере. Никаких скриншотов, облака и аккаунтов.

## Установка

1. Откройте вкладку **Releases** и скачайте установщик последней версии
2. Windows покажет «Неизвестный издатель» → **Подробнее → Выполнить в любом случае**
3. Укажите папку с рисунками — и просто работайте

## Обновления

Приложение само проверяет новые версии: **Настройки → Проверить обновления**.
